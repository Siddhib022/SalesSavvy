# 🚀 SalesSavvy Frontend

SalesSavvy is a modern web application designed to manage sales activities and streamline customer interactions. This repository contains the **frontend of the SalesSavvy application**, built using modern web technologies to provide a responsive and user-friendly interface.

The application allows users to interact with the system through an intuitive UI, manage sales data, and communicate with the backend services.

# ⚙️ Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Siddhib022/SalesSavvy.git
```

### 2️⃣ Navigate to the project folder

```bash
cd SalesSavvyFrontEnd-main
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the development server

```bash
npm start
```

The application will run on:

```
http://localhost:3000
```
